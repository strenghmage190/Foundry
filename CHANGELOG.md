# Changelog

All notable changes to the Beyonders system will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-30

### Added
- Initial system implementation for Foundry VTT
- Actor types: Hero, Villain, Pawn
- Item types: Weapon, Spell
- Resource management (Health, Power)
- Custom data models with proper validation
- Handlebars templates for actor and item sheets
- CSS styling with Beyonders theme
- Integrated React web app support
- Localization support (English, Portuguese-BR)
- Token attribute tracking
- System settings and configuration

### Technical
- Custom Actor and Item document classes
- Data model inheritance and migration support
- Proper schema validation
- Foundry VTT v11+ compatibility
- ES6 modules implementation
- Following Foundry VTT system development best practices
